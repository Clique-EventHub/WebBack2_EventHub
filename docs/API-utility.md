# **Utility**

---
## get data 
get fields of data

* **URL**

  `/reqdata` 

* **Method:**

  `GET`

*  **URL Params**

    `None`

* **Data Params**

    `fields=requestableFieldUser`

* **Success Response:**

  * **Code:** `2xx`

    **Content:**
		{ field : ... }

* **Error Response:**

  * **Code:** 500

    **Content:**
    {"msg" : "error"}
